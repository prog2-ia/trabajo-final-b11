¡Bienvenido a nuestra biblioteca musical!

Este es nuestro proyecto de Programación II basado en una Biblioteca Musical
en python, que se centrará en simular el funcionamiento de aplicaciones como 
Spotify o Apple Music mediante el uso de clases, métodos y objetos.

En esta primera evaluación cruzada, nuestro objetivo ha sido el de implementar
herencia, métodos abstractos y manejo de errores con property y setter.

Por tanto, hemos obtenido una primera versión de nuestro trabajo que permite
guardar canciones, ver álbumes y artistas, además de crear listas, sumarlas y
compararlas.

Para instalar el programa, solo se debe descargar los archivos de este
repositorio y ejecutar el proyecto en python, el cual debe estar instalado
de antemano en tu equipo.
